Manual
==================

Este programa ten a función de xestionar unha base de datos mediante unha interface de usuario.
Na primeira pantalla pedirásenos iniciar sesión introducindo nome de usuario e contrasinal, unha vez introducidos uns datos válidos debemos  selecionar a que pantalla á que accederás, ``Xestión de usuarios`` ou ``Xestión de produtos``.

Na ventana de ``Xestión de Clientes`` atoparemos unha lista cos datos de todos os clientes, así como uns botóns para interactuar cos datos, se seleccionamos un cliente, os seus datos pasaran aos campos de abaixo, donde poderemos editar, para subir os cambios deberemos pulsar o botónde ``Consulta``.
Co botón ``Insertar`` podemos añadir un cliente tras rechear os campos vacíos.
Co botón ``Borrar`` eleminaremos da base de datos ao cliente seleccionado.

Na ventana de ``Xestión de Produtos`` teremos o botón de ``Xerar informe``, co que se nos creará un informe en PDF de todos os produtos da base de datos no escritorio.
